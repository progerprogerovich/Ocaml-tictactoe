# Сетевые Крестики-Нолики на OCaml (Lwt)

Асинхронный PVP-сервер для игры в крестики-нолики через терминал.

### Запуск сервера

Для компиляции и запуска требуются установленные `ocaml`, `findlib` и библиотека `lwt`.

```bash
# Компиляция (в зависимости от вашей настройки сборки)
ocamlfind ocamlc -package lwt.unix -linkpkg server.ml -o ttt_server

# Запуск сервера (по умолчанию порт 9000)
./ttt_server
```
