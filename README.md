# Сетевые Крестики-Нолики на OCaml (Lwt)

Асинхронный PVP-сервер для игры в крестики-нолики через терминал.

## 🛠 Установка окружения

Перед сборкой проекта необходимо развернуть компилятор OCaml и установить зависимость `Lwt`. Для автоматической настройки запустить готовый скрипт:

```bash
./setup.sh
```
